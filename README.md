# VocalChords
To make your own version follow these steps:
1 - Go on TModLoader

2 - Select Workshop

3 - Select Develop Mods

4 - Select Create Mod

5 - Fill the fields

6 - Open your mode files by clicking the button with the white square in it (near your mod's name)

7 - Add the Player.cs file

8 - Line 9 will be 'namespace VocalCords'. Change this to 'namespace [Your mod's name]' (not DisplayName)

9 - Read the comments and you should be good to go.

For help clancini on Discord or the tModLoader discord server

You can republish this mod with your changes, just give me some credit or something thanks :) <3
